# Smart Home Configurator
Приложение для 3D конфигурирования умного дома. Позволяет создавать план здания, расставлять мебель и устройства, управлять освещением и датчиками.

## Возможности

- **2D Редактор плана** - Рисование комнат, стен, дверей и окон
- **3D Визуализация** - Реалистичный 3D вид с управлением камерой
- **Библиотека мебели** - Процедурные 3D модели мебели, ламп и датчиков
- **Умное размещение** - Автоматическое размещение на стенах и потолке
- **Проверка коллизий** - Мебель не пересекается со стенами и другими предметами
- **Управление проектами** - Сохранение и загрузка проектов
- **База данных** - Хранение проектов в H2
- **Генерация текстур** - Процедурные текстуры для стен, пола и дверей

## Управление

### 2D Редактор
- Клики по углам - создание комнат
- R/Shift - поворот предмета перед размещением
- Средняя кнопка мыши - панорамирование
- Колесико - масштабирование
- Выбор объекта - редактирование свойств
