# Smart Home Configurator

Курсовая работа по объектно-ориентированному программированию.

## Описание проекта

Программа для конфигурирования и симуляции системы умного дома с 3D визуализацией.

## Функциональность

- 2D и 3D рендеринг плана дома
- Различные режимы симуляции (случайная, умный дом, ручное управление)
- Сохранение и загрузка проектов в базу данных H2
- Управление освещением (включение/выключение ламп)
- Орбитальная камера и режим ходьбы


## Установка и запуск

1. Клонируйте репозиторий
2. Соберите проект: `mvn clean compile`
3. Запустите: `mvn javafx:run`

## 3D модель

Для работы 3D режима необходимо скачать модель дома:

1. Скачайте модель:
2. Поместите файл в папку `src/main/resources/models/`
3. Поместите текстуры в папку `src/main/resources/models/textures/`
4. Запустите программу

Если модель не найдена, программа запустится в упрощенном режиме. Модель можно загрузить через кнопку "ЗАГРУЗИТЬ" в интерфейсе.
